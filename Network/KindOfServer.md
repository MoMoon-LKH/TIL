# 서버 운영 방식
## 온프레미스 방식
- 직접 서버를 운용 관리하는 방식

## 클라우드 서비스
- 서버를 직접 관리 X, 인터넷을 통해 서버의 기능만 이용할 수 있는 방식

### 클라우드 서비스 분류
#### 1) IaaS
- 서버의 CPU와 메모리, 저장장치와 같은 하드웨어 부분을 사용 가능
#### 2) PaaS
- 서버의 OS와 OS 상에서 동작하는 데이터베이스 등의 미들웨어을 포함한 부분을 사용 가능
#### 3) SaaS
- 서버의 특정 소프트웨어 기능을 사용 가능
